Function smarty_function_es

an escape modifier to escape html and script tags in html can use:

{es v="<strong>FooBar</strong>"} or {es value="<strong>FooBar</strong>"} or {es v="<strong>FooBar</strong>" assign=fooBar}

The last example will assign the smarty variable $fooBar

Output: <strong>FooBar</strong>
